## Description
抽奖应用：
初始：n 个人参与抽奖，m 个奖品。
每次抽1 个奖品，ni 人抽到mj。则：
抽奖人池除去 ni, 总人数 n-1
奖品池除去mj，总奖池数 m-1
若抽奖人池先到 0，则抽奖人池重新至为 n，抽奖人全部重新加入抽奖人池
若奖池数先到 0，则抽奖结束
## Requirements
1. 抽奖人名，奖品 通过一个json文件配置
2. 抽奖按钮点击后有人名持续变动的动态效果
3. 使用公平公正的随机算法
4. 不依赖数据库，在windows机器或mac机器的chrome浏览器直接能运行
