# 抽奖工具使用说明

## 启动方式

### 方法一：使用Python内置服务器（推荐）

1. 打开终端或命令行工具
2. 进入项目目录：
   ```bash
   cd /path/to/toughguy
   ```
3. 启动本地服务器：
   - Python 3.x:
     ```bash
     python3 -m http.server 8000
     ```
   - Python 2.x:
     ```bash
     python -m SimpleHTTPServer 8000
     ```
4. 打开浏览器，访问 `http://localhost:8000`

### 方法二：直接打开HTML文件

1. 直接双击 `index.html` 文件在浏览器中打开
2. 注意：如果直接打开文件，可能因为浏览器安全策略无法加载JSON配置文件，建议使用方法一

### 方法三：使用其他Web服务器

你也可以使用任何Web服务器（如Apache、Nginx等）将项目部署到服务器上访问。

## 使用步骤

1. 点击"📁 选择配置文件"按钮
2. 选择JSON格式的配置文件（参考项目中的 `config.json` 示例）
3. 选择文件后会自动加载配置
4. 点击"开始抽奖"按钮开始抽奖
5. 点击"停止"按钮结束当前抽奖，系统会自动选出中奖者
6. 中奖历史会显示在页面下方
7. 可随时点击"重置"按钮重置整个抽奖过程

## 配置文件格式

配置文件必须是JSON格式，包含两个数组：
- `people`: 参与抽奖的人员姓名列表
- `prizes`: 奖品列表

示例：
```json
{
  "people": [
    "张三",
    "李四",
    "王五"
  ],
  "prizes": [
    "一等奖：iPhone 15",
    "二等奖：iPad Air",
    "三等奖：AirPods"
  ]
}
```

## 功能说明

- **抽奖规则**：每次抽取一个奖品，中奖者从抽奖池中移除，奖品也从奖池中移除
- **人员池重置**：当抽奖人池先变为0时，会自动重置为原始人员列表
- **抽奖结束**：当奖品池先变为0时，抽奖结束
- **实时显示**：显示当前抽奖人数、剩余奖品数、当前抽奖人列表和剩余奖品列表
- **中奖历史**：记录所有中奖者信息
- **动画效果**：抽奖过程中人名会快速滚动显示
